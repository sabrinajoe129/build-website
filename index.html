WHAT IS HTML?

HTML stands for HyperText Markup Language. Basically, a markup language is a text-encoding system which specifies the structure and formatting of a document and the relationship among its parts. Markup function is to control the display of a document or even enrich its content to facilitate automated processing. In short, HTML tells the browser how to display content like images, text and links.
It usually used in the making of web pages such as blogs, business websites, personal websites, news websites and web apps.

FUN FACT

HTML was created by Tim Berners-Lee in 1991.
Tim was a scientist working at CERN, a research center in Switzerland

WHAT YOU WILL NEED:

  1. Text editor (PC: Notepad, Mac: TextEdit)

STEPS TO CREATE YOUR OWN WEBSITE

Step 1:Open Notepad

Type notepad 👉 Hit ENTER

Step 2:Type your HTML code


  <!DOCTYPE html>
    <html>
      <head>
        <title>My First Website</title>
      </head>
      <body>
         <h1>Hello, World!</h1>
         <p>This is my first web page using HTML.</p>
      </body>
    </html>

What you should know

<!DOCTYPE html> Declare the document as HTML5
<html> Root element of an HTML page
<head> Contain meta information
<title> Element specifies a title for the document
<body> Everything that display on the page
<h1> Main heading
<p> Paragraph

Step 3:Add more elements

LINK


<a href="https://www.google.com">Visit Google</a>

IMAGE


 <img src="image.jpg" alt="description" width="300">

LIST

“ul” is for Unordered list or bullet


<ul>
    <li>Learn HTML</li>
    <li>Practice daily</li>
    <li>Build cool stuff</li>
</ul>

“ol” is for ordered list or numbered list.


<ol>
    <li>Learn HTML</li>
    <li>Practice daily</li>
    <li>Build cool stuff</li>
</ol>
